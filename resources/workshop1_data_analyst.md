## 故事背景

周一早上九点，你刚到公司，投资总监就走过来拍了拍你的桌子：

> "我们基金要开始布局 AI 赛道了。我手上有一份全球 35 家 AI 独角兽公司的数据，你帮我做一份全景分析报告——国家分布、赛道对比、资本效率、谁被高估谁被低估，最好再结合最新的行业动态。下午三点投委会要用。"

你看了一眼时间，还有 6 个小时。打开 Excel？写 Python 脚本？做 PPT？光是数据清洗和画图就要大半天……

这时你想起了 OpenClaw。

你决定试试：不写一行代码，只用自然语言对话，看看能不能在 30 分钟内搞定这份报告。

以下就是你接下来要做的事。

## 实验 1：一键数据概览

在本实验中，你将体验 OpenClaw 如何读取数据文件并自动完成数据分析。全程无需编写任何代码。

### 步骤 1：确认数据访问权限

在 OpenClaw 对话框中发送以下内容：

```
确认你是否可以读取 s3://openclaw-user-files-<实验环境aws account id>-us-west-2/<实验学员个人命名空间>/workshop_data/ 下的文件。请列出文件名。
```

你应该看到以下文件被列出：
- SKILL_data_analyst.md
- SKILL_report_gen.md
- ai_unicorns.csv

### 步骤 2：安装 Skill

```
帮我安装这两个skill
```

### 步骤 3：发送分析请求

```
请读取 ai_unicorns.csv 文件，给我一个全面的数据概览：
- 数据基本情况（行数、列数、数据类型）
- 各字段的统计摘要
- 国家分布、赛道分布
- 找出估值 Top 5 公司
- 找出增长最快的 5 家公司
```

### 你将看到

OpenClaw 会自动读取 CSV 文件，生成并执行 Python 分析代码，最终输出结构化的数据摘要。你没有写任何一行代码，但完成了本来需要约 20 行 Python 才能实现的工作。

## 实验 2：多维可视化分析

接下来，让 OpenClaw 帮你把数据变成直观的图表。

### 发送以下内容

```
基于刚才的数据，帮我生成 4 张分析图表，保存为 markdown：

1. 各国 AI 独角兽数量和总估值对比（分组柱状图）
2. 赛道分布饼图（按公司数量）
3. 估值 vs 融资额 散点图（用颜色区分国家，用气泡大小表示员工数）
4. 各公司同比增长率 Top 10 水平条形图

要求：深色主题，专业配色，图表标题用中文
```

### 你将看到

OpenClaw 会以图表的形式呈现 4 组分析结果。你可以要求openclaw 把图片上传到S3中，并且生成下载链接供你查看。

如果对某张图表不满意，试试追加一句话来迭代，例如：「把第 3 张图表加上具体数值标注」。

## 实验 3：深度交叉分析

这一步我们让 OpenClaw 做更有深度的分析，不只是跑数据，还要给出判断和洞察。

### 发送以下内容

```
帮我做更深入的分析：

1. 计算每个国家的"资本效率"（收入/融资总额），哪个国家的 AI 公司最会赚钱？
2. 2023 年成立的公司 vs 2019 年以前的公司，估值和增长率有什么差异？新公司是不是估值泡沫更大？
3. 中国 vs 美国的 AI 公司全方位对比（估值、融资、收入、员工数、盈利比例）
4. 找出"被低估"的公司：收入高但估值相对低的

把分析结果用表格展示，配上你的判断和洞察。
```

### 你将看到

OpenClaw 会输出多张交叉分析表格，并附上带有观点的分析洞察。注意它不只是呈现数字，还会给出"被低估的公司"这样的判断性结论。

想一想：你觉得 Agent 的判断靠谱吗？

## 实验 4：联网搜索增强分析

到目前为止，我们的分析都基于本地数据。现在让 OpenClaw 联网获取最新信息，看看数据是否已经过时。

### 发送以下内容

```
帮我搜索一下最近 3 个月 AI 行业的重大融资事件和趋势变化，
然后结合我们的数据集，回答：

1. 我们数据里的估值信息是否已经过时？哪些公司可能有了新的融资？
2. 2025-2026 年 AI 投资的整体趋势是升温还是降温？
3. 有没有我们数据里没包含的新晋独角兽值得关注？

把搜索到的最新信息和我们的数据结合起来分析。
```

### 你将看到

OpenClaw 会联网搜索实时信息，然后将搜索结果与本地数据集结合对比分析。分析不再是"闭门造车"，而是融合了多个信息来源。

> 备注：如果网络搜索不可用，可以将请求改为「根据你的知识，补充分析 2025 年 AI 行业趋势」作为替代。

## 实验 5：一键生成完整分析报告

最后一步，让 OpenClaw 把前面所有的分析成果汇总成一份专业报告。

### 发送以下内容

```
把前面所有的分析汇总，生成一份完整的 Markdown 分析报告并保存到S3。

报告要求：
- 标题：《2026 全球 AI 独角兽投资全景分析》
- 包含：摘要、数据概览、图表（引用之前生成的）、深度分析、行业趋势、投资建议
- 风格：专业分析师报告，中文为主
- 最后附上数据来源说明和免责声明
```

### 你将看到

OpenClaw 会生成一份 2000-3000 字的完整 Markdown 分析报告，嵌入之前生成的图表引用，包含摘要、分析、结论和建议。从S3下载这个文档（可以让OpenClaw帮你生成下载链接）

回顾一下：我们只发了 5 条消息，就完成了数据加载、分析、可视化、联网研究、报告生成的全流程。

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| 图表中文显示乱码 | 告诉 OpenClaw：「图表中文乱码了，请修复字体设置」 |
| 图表生成失败 | 告诉 OpenClaw：「图表生成失败了，请换一种方式重试」 |
| 网络搜索不可用 | 将实验 4 的请求改为「根据你的知识分析 AI 行业趋势」 |
| 回复内容太长 | 追加一句：「简洁回复，每个分析点不超过 3 句话」 |
| CSV 文件读取失败 | 确认文件路径是否正确，或请求讲师协助 |
